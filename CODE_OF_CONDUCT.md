# Code of Conduct

## Our pledge

We pledge to make participation in Caesar Harness Agent welcoming, respectful, and productive for everyone.

## Expected behavior

Participants are expected to:

- use welcoming and inclusive language
- respect differing viewpoints and experience levels
- give constructive feedback focused on the work
- accept responsibility and apologize when mistakes happen
- prioritize project quality, safety, and maintainability

## Unacceptable behavior

Unacceptable behavior includes:

- harassment, insults, threats, or discriminatory language
- sexualized language or imagery
- trolling, sustained disruption, or bad-faith argument
- publishing private information without explicit permission
- abuse of project channels, issue templates, or security reporting paths

## Enforcement

Report unacceptable behavior to Caesar Nexus Labs maintainers through the private channel listed on the GitHub repository or organization profile. Do not post sensitive conduct reports publicly.

Maintainers may remove comments, close issues, restrict participation, or ban contributors when needed to protect the project and community.

## Scope

This Code of Conduct applies in project spaces, public community channels, private maintainer interactions, and any context where someone represents the project.
